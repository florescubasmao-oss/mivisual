CONFIGURACIÓN AUTOMÁTICA
Al iniciar sesión con un usuario cuyo perfil sea GERENCIA LIMA, Apps Script sincroniza sus filas en PERMISOS_MODULOS tomando como base Jefatura y aplicando las restricciones indicadas.

INSTALACIÓN
1. Reemplazar únicamente los archivos incluidos respetando las carpetas.
2. Copiar Code.gs en Apps Script.
3. Guardar y crear un NUEVO DEPLOY de Apps Script.
4. Actualizar la URL del frontend solo si el despliegue genera una URL distinta.
5. Subir los archivos frontend a GitHub y hacer commit.
6. Iniciar sesión con GERENCIA LIMA para sincronizar permisos.
