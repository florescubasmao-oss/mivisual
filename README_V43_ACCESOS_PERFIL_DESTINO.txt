MI VISUAL v43 - Accesos por destino y perfil

Cambio aplicado:
- La hoja de ACCESOS ahora usa columnas:
  A: ID
  B: DESTINO
  C: PERFIL
  D: NOMBRE
  E: LINK

Reglas:
- DESTINO puede ser TODOS, una sede, varias sedes separadas por coma o una cuadrilla exacta.
  Ejemplo: CHICLAYO,PIURA
- PERFIL puede ser TODOS, TECNICO, SUPERVISOR, JEFATURA, ADMIN o varios separados por coma.
  Ejemplo: SUPERVISOR,JEFATURA,TECNICO
- El link se muestra solo si cumple DESTINO + PERFIL.
- ADMIN también puede visualizar accesos configurados como JEFATURA.
